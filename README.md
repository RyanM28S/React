## ✨ Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **Autenticação JWT** | Login e cadastro seguro utilizando JSON Web Tokens e criptografia de senhas com Bcrypt. |
| **Avaliação de Docentes** | Interface para visualização detalhada e feedback sobre o corpo docente. |
| **Gestão de Setores** | Monitoramento e avaliação de diferentes departamentos da instituição. |
| **Painel do Professor** | Área exclusiva para registro de notas e acompanhamento de desempenho dos alunos. |
| **Interface Responsiva** | Design moderno e adaptável para diferentes dispositivos, desenvolvido com Sass. |

## 🛠️ Tecnologias Utilizadas

### Front-End
- **React 19**: Biblioteca principal para construção da interface.
- **Vite**: Build tool ultra-rápida para o desenvolvimento.
- **Sass (SCSS)**: Pré-processador CSS para estilização avançada e modular.
- **Framer Motion**: Biblioteca para animações fluidas e interativas.
- **React Router Dom**: Gerenciamento de rotas e navegação.
- **React Hot Toast**: Notificações elegantes para feedback ao usuário.

### Back-End
- **Node.js & Express**: Ambiente de execução e framework para a API.
- **MariaDB**: Banco de dados relacional para persistência de informações.
- **Bcrypt**: Hashing de senhas para segurança máxima.
- **JSON Web Token (JWT)**: Autenticação baseada em tokens.

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js instalado
- Banco de Dados MariaDB/MySQL ativo

### 1. Configurar o Banco de Dados
Execute o script localizado em `/DataBase/Opina_create.sql` no seu gerenciador de banco de dados.

### 2. Iniciar o Back-End
```bash
cd BackEnd
npm install
npm start
```
*O servidor rodará em `http://localhost:3001`*

### 3. Iniciar o Front-End
```bash
cd FrontEnd
npm install
npm run dev
```
*Acesse `http://localhost:5173` no seu navegador.*
