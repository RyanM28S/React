# ✨ Sobre o Projeto

O **Salotti Opina** é uma plataforma escolar criada com o objetivo de permitir avaliações construtivas sobre:

* 👨‍🏫 Professores
* 🏫 Setores da escola
* 📊 Experiências institucionais
* 💬 Feedbacks anônimos
* 📈 Painéis e estatísticas

O sistema foi pensado para oferecer uma experiência visual moderna, com animações fluidas, interface premium e foco total em usabilidade.

# ⚙️ Como Rodar o Projeto

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/seuusuario/salotti-opina.git
```

---

## 2️⃣ Entre na pasta

```bash
cd salotti-opina
```

---

## 3️⃣ Instale as dependências

```bash
npm install
```

---

## 4️⃣ Rode o projeto

```bash
npm run dev
```
